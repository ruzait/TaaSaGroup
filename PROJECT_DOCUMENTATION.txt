                        4. SEO IMPLEMENTATION GUIDE
================================================================================

4.1 ON-PAGE SEO
─────────────────────────────────────────────────────────────────────────────

META TAGS IMPLEMENTED:
- <title>: "TaaSa Rice Processing Company | Premium Rice Producer & Exporter Sri Lanka | ISO 9001 Certified"
- <meta name="description">: Comprehensive 160+ character description
- <meta name="keywords">: 50+ rice-related keywords
- <meta name="robots">: index, follow with advanced directives
- <link rel="canonical">: Points to primary URL

HEADING STRUCTURE:
- H1: Company tagline/hero headline
- H2: Section titles (Features, About, Products, etc.)
- H3: Subsection titles within sections
- Proper hierarchy maintained throughout

IMAGE SEO:
- All images have descriptive alt attributes
- Lazy loading implemented (loading="lazy")
- Proper aspect ratios maintained

URL STRUCTURE:
- Clean, human-readable URLs
- Section anchors for SPA navigation (#products, #contact)
- No dynamic query parameters

4.2 TECHNICAL SEO
─────────────────────────────────────────────────────────────────────────────

XML SITEMAP (sitemap.xml):
- Contains all pages and sections
- Image URLs included for Google Image search
- Last modified dates
- Priority and change frequency
- Location: /sitemap.xml

ROBOTS.TXT:
- Allows all crawlers
- Disallows private areas (/api/, /admin/)
- Points to sitemap
- Location: /robots.txt

STRUCTURED DATA (JSON-LD):

a) LocalBusiness Schema:
- Company name, address, phone, email
- Geographic coordinates
- Opening hours (24/7)
- Price range
- Service area

b) Organization Schema:
- Founder information
- Certifications (ISO 9001, ISO 22000, GMP)
- Area served (Sri Lanka, Middle East, Africa)

c) Product Schema:
- White Rice product details
- Non-Basmati Rice product details
- SKU, GTIN, brand, manufacturer
- Availability status

d) FAQPage Schema:
- 5 common questions and answers
- Improves SERP appearance

e) BreadcrumbList:
- Navigation structure

PWA SUPPORT:
- manifest.json for installable app
- Mobile-optimized viewport
- Touch-friendly design

================================================================================
                        14. TROUBLESHOOTING & FAQ
================================================================================

Q: Build fails with errors?
A: 
  1. Clear node_modules: rm -rf node_modules
  2. Reinstall: npm install
  3. Try build again: npm run build

Q: Images not loading?
A: 
  1. Check public/assets/ folder exists
  2. Verify file paths in code
  3. Check browser console for 404s

Q: Animations not working?
A: 
  1. Check Framer Motion is imported
  2. Verify React version compatibility
  3. Check browser supports CSS animations

Q: Mobile menu not working?
A: 
  1. Check Navbar component for state
  2. Verify CSS classes for mobile menu
  3. Test with responsive mode in dev tools

Q: SEO not showing in Google?
A: 
  1. Submit sitemap to Search Console
  2. Wait 1-2 weeks for indexing
  3. Check meta tags in page source
  4. Verify robots.txt allows crawling

Q: WhatsApp link not working?
A: 
  1. Check phone number format
  2. Verify whatsappClean in constants.js
  3. Test on actual device
